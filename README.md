# 🚀 Vault Zero to Hero

Learn **HashiCorp Vault** from scratch to production with a structured 21-day roadmap.

This repository contains all the hands-on demos, notes, and examples used in the YouTube playlist.

## 🎯 Goal of This Series

By the end of this series, you will:

* Understand secrets management fundamentals
* Work with Vault CLI, API, and UI
* Implement authentication & policies
* Use dynamic secrets and leasing
* Integrate Vault with applications
* Deploy Vault in real-world environments

---

## 🧰 Prerequisites

* Basic Linux knowledge
* Basic understanding of DevOps concepts
* Docker (optional but recommended)
* Kubernetes (for advanced sections)
