## What is HashiCorp Vault?

HashiCorp Vault is a secrets management tool used to securely store, access, and manage sensitive data.

Examples of secrets:

- Database passwords
- API keys
- Tokens
- Certificates
- Cloud credentials
- Application credentials

Vault helps DevOps, cloud, and platform teams avoid hardcoded secrets and manage access securely across applications, CI/CD pipelines, Kubernetes, and cloud environments.
